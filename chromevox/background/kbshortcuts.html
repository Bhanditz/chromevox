ChromeVox Keyboard shortcuts

Note: Search + Shift are the modifier keys. While holding down Search and Shift, press the subsequent keys in order, one at a time. For example, to execute the shortcut "Search + Shift - N, H", press N followed by H, while holding down Search and Shift keys

Ctrl or Search : Stop speaking
Search, Search : Enable/Disable sticky mode
Tab : Jump to next focusable item
Shift + Tab : Jump to previous focusable item
Search + Shift - Up arrow : Navigate backward
Search + Shift - Down arrow : Navigate forward
Search + Shift - Left arrow : Decrease navigation granularity
Search + Shift - Right arrow : Increase navigation granularity
Enter : Take action on current item
Search + Shift - Space : Click on current item
Search + Shift - Period : Show ChromeVox help
Search + Shift : Hide ChromeVox help
Search + Shift - Forward slash : Toggle search widget
Search + Shift - O, B : Open bookmark manager
Search + Shift - O, W : Open options page
Search + Shift - O, K : Open keyboard explorer
Search + Shift - N, A : Switch to next TTS engine
Search + Shift - Dash : Decreaste rate of speech
Search + Shift - Equal sign : Increase rate of speech
Search + Shift - Semicolon : Decrease pitch
Search + Shift - Single quote : Increase pitch
Search + Shift - Open bracket : Decrease speech volume
Search + Shift - Close bracket : Increase speech volume
Search + Shift - T, Enter : Enter table
Search + Shift - T, Escape : Exit table
Search + Shift - T, Up arrow : Go to previous table row
Search + Shift - T, Down arrow : Go to next table row
Search + Shift - T, Left arrow : Go to previous table column
Search + Shift - T, Right arrow : Go to next table column
Search + Shift - T, Comma : Go to beginning of table
Search + Shift - T, Period : Go to end of table

Search + Shift - P, 1 : Previous level 1 heading
